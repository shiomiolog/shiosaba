# Shiosaba 🐈🐟

An ongoing personal project to develop a toy web browser, based on the book "[［作って学ぶ］ブラウザのしくみ](https://gihyo.jp/book/2024/978-4-297-14546-0)".

### 📌 Overview
This repository is my personal playground for exploring the inner workings of web browsers. Inspired by the "Saba" (mackerel) project and the drift of the tide, this project dives into the ocean of the Web to understand how browsers process and render information.

The project name "Shiosaba" is a pun on the book's nickname ("Saba-book") and a twin project to my OS, **shiosabi**.

### 📖 Reference
- **Sample Code:** [d0iasm/sababook](https://github.com/d0iasm/sababook) (Official sample code for the book)
- **Latest Official Code:** [d0iasm/saba](https://github.com/d0iasm/saba) (Author's latest implementation)
